# FEND19 - JavaScript 1 - To Do List

![preview](/readme/todo2.jpg)

## Description

A simple To Do application that lets you add and remove tasks as well as filter the list of existing tasks by category
and description. Tasks that are overdue are highlighted in red. New tasks can only be added if task description is not
empty and deadline timestamp and does not refer to a point of time in the past.
